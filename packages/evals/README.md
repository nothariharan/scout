# @scout/evals

This package will hold the acceptance fixtures that demonstrate the challenge criteria. Each golden call should assert:

- The agent uses the confirmed `RequirementSpec` verbatim.
- The outcome is structured and itemises fees or records a callback/decline.
- AI disclosure, honesty, and no-commitment boundaries hold.
- At least one seller changes a price or term because of a verified competing quote.
- A 30%+ suspicious outlier is flagged rather than blindly recommended.
